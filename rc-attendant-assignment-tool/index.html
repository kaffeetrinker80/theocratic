Was dieses Tool können soll

Dieses Tool ist für eine komplette Ordner-Einteilung eines 3-Tage-Kongresses gedacht: Gesamtplan, flexible Positionen, flexible Zeitblöcke, Ordnerstamm, Druck und persönliche Einzelübersichten.

Empfohlener Ablauf

1. Grunddaten pflegen: Titel, Ort und Tage unter „Konfiguration“ anpassen.

2. Zeitblöcke definieren: Die Zeiten können komplett frei gesetzt werden. Dadurch ist das Tool nicht an eine bestimmte Kongressstätte gebunden.

3. Positionen anlegen: Jede Position hat Nummer, Bereich, Beschreibung und Soll-Besetzung. Das Tool markiert leere oder unterbesetzte Felder.

4. Ordnerstamm einfügen: Namen einmal pflegen. Danach in der Gesamtübersicht per Dropdown einteilen.

5. Gesamtübersicht drucken: Für A4 Querformat optimiert. Bereichsfilter hilft, einzelne Abschnitte auszudrucken.

6. Einzelübersicht nutzen: Namen anklicken. Dort sieht der Ordner alle Einsätze sortiert nach Tag und Zeit. Text kann direkt für WhatsApp kopiert werden.

Datenschutz

Alle Daten bleiben lokal im Browser. Für Sicherung oder Weitergabe bitte JSON Export/Import verwenden. Für GitHub können echte Namen draußen bleiben und nur per JSON geladen werden.
