Help – A/V, Streaming & Baptism Planner

Quick Start (3 steps)
  1. Role Management: enter name pools (comma-separated) for each role.
  2. Shift Plan: choose Day → choose Area tab → add shifts and assign brothers.
  3. Teamwork: Export JSON → Import JSON in master file → select only the area to merge.

What this tool is for

Maintain role pools and plan shifts for: Program Audio Video Stage Streaming (IT) plus a separate planning view for Baptism.

Main sections

1) Role Management (Pools)

  • Enter names as a comma-separated list for each role.
  • These names become options in Shift Plan dropdowns.
  • Helper roles use a bigger input area (good for 10–15 names).

2) Shift Plan – A/V Shifts

  • Day tabs: Friday / Saturday / Sunday
  • Area tabs: Program / Audio / Video / Stage / Streaming (IT)
  • Helpers are flexible: add/remove helper dropdowns per shift row.

3) Shift Plan – Baptism

  • No time slots.
  • Assign a brother and write a short responsibility note (pool / stairs / changing rooms / towels / doors, etc.).
  • Add as many Baptism Helpers as needed (each with a name + description).

Saving & data persistence

  • The tool auto-saves to your browser local storage.
  • Re-open the file in the same device + same browser to restore data.
  • For another device/browser, use Export / Import.

Export (JSON)

  • Export includes: meta (title/subtitle), rolesConfig, scheduleData, baptismData.
  • The filename timestamp uses your local time.

Import (Merge by Area)

Import is a merge, not a full overwrite. After selecting a JSON file, you choose what to merge.

How shifts are matched

Day + Start Time + End Time

Meta (title/subtitle) import

  • Title/subtitle are imported only if you tick Import title/subtitle (meta).

Recommended teamwork workflow

  1. Each team works on its own area tab (e.g., Video).
  2. Export JSON.
  3. Coordinator imports and selects only that area in the merge dialog.
  4. Repeat for other areas.

Printing

  • Print (current view) prints what you currently see.
  • Switch to Baptism tab to print Baptism.
