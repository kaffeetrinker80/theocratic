PTC Tool – EU Public Speakers · Quick guide for v2.0.7
What this tool is for
This tool helps Public Talk Coordinators work with the EU Public Speakers list faster and more comfortably. It can import the CSV export, search speakers and congregations, save local notes and memory, mark favorites, calculate distance from your home congregation when geo data is available, and prepare correction suggestions for Ronald.
CSV import Speaker + congregation view Favorites + notes Distance + radius filter JSON backup
1. Import and update workflow
Start with Import CSV. The tool reads congregation data, speaker names, contact details, circuits, meeting times, outlines, and optional geo fields.

When you get an updated CSV, just import it again. Speaker memory such as favorites, notes, last invited, and last used can still be matched again when the speaker is recognized.

Use Export JSON from time to time so your personal work is backed up.
2. Top filters – first row
The first filter row is for the normal list work:
  • main search
  • outline filter
  • country
  • circuit
  • meeting day
  • speaker role
The main search looks through speaker name, congregation, city, country, circuit, phone, email, outline number, outline title, and saved local notes.
3. Distance tools – second row
The second filter row is only for distance handling:
  • Distance mode = Air-line or Estimated drive × 1.25
  • Max distance = for example within 50 / 100 / 150 / 200 km
  • Sort = Name or Distance
These controls work when the imported CSV contains usable geo coordinates for the congregations.
4. Home congregation
Open a congregation and click Set as home congregation.

After that:
  • the top button Home Congregation jumps back to your congregation
  • distances are calculated from that point
  • result cards can show nearby congregations and speakers
If no home congregation is set, the distance tools stay available but cannot calculate meaningful values yet.
5. Speaker View and Congregation View
Speaker View is best when you already know the brother or the outline you need.

Congregation View is best when you want to review a whole congregation first, including the PTC and all available speakers in one place.

In congregation view you can click a speaker directly in the speaker list and open his speaker details on the right side.
6. Back to congregation
When you open a speaker directly from a congregation, the tool remembers that congregation.

In the selected speaker area you will see ← Back to congregation directly above the highlighted speaker box. That makes it easy to review several brothers from the same congregation without searching again.
7. Outline search
The outline filter can be used in two ways:
  • by number, for example 76
  • by words from the title, for example family, hope, or hospitality
This makes it easy to find brothers even when you only remember the subject but not the exact outline number.
8. Favorites and speaker memory
For each speaker you can save:
  • favorite status
  • last invited
  • last used
  • notes
This is local browser memory and is also included in the JSON export. It is useful for brothers you invite often, brothers you want to keep in mind, or to avoid using the same speaker too frequently.
9. Contacts and quick actions
The tool turns saved data into practical shortcuts:
  • e-mail links
  • phone / WhatsApp links where possible
  • hall address to Google Maps
  • PTC contact export as .vcf
This is meant to reduce copy and paste work.
10. Corrections and Help Us Improve
If something in the list is outdated or incomplete, use the correction section on a speaker or congregation.

You can describe what is wrong, suggest the correction, and send a prepared WhatsApp or email message.

The page Help Us Improve also collects likely field issues in one place for easier review.
11. Best practical routine
A simple workflow that works well:
  1. import the newest CSV
  2. set or confirm your home congregation
  3. use search and outline filter
  4. switch to congregation view when you want to compare several brothers from one congregation
  5. use distance filters if you want nearby options first
  6. keep notes and memory updated
  7. export JSON as backup
12. Small reminder
The tool can make the list much easier to work with, but the results still depend on the quality of the imported data. So careful review, timely updates, and useful corrections remain very valuable.
