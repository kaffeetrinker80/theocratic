Help & workflow

1. First setup

Choose the language at the top. Edit title, subtitle and location directly in the header by clicking the text. Then open Configuration and enter the global contact details.

  • Global contact is used for the full printed plan and email text.
  • Header text is saved automatically after editing.

2. Congregations

Add each congregation once with name, number of cleaning groups, contact person and email address. The group number controls the automatic rotation.

  • Example: 4 groups means this congregation receives four Saturdays in a row.
  • Use Save changes after editing an existing congregation.
  • A congregation cannot be removed while it is still used in the current plan.

3. Create an annual plan

Select the year. If it is missing, use + Year. Under Configuration, set the first cleaning date for that year. It must be a Saturday.

  • Click Complete annual plan to fill all remaining Saturdays.
  • Existing entries remain; only missing Saturdays are added.
  • Reset year deletes all cleaning dates for the selected year.

4. Work with dates

Use Add new date for single entries. In the table you can assign open dates, edit entries, swap two dates or delete an entry.

  • Jobcards such as window cleaning are stored as notes for that date.
  • Additional notes are shown in prints, emails and exports.
  • The overview cards show how many dates each congregation has in the selected year.

5. Print, email and backup

Print full plan creates the yearly overview. Print congregation creates a plan for one congregation only. Send email opens your mail program with the dates for that congregation.

  • Export data creates the JSON backup file.
  • Import data restores a JSON backup and overwrites the current local data after confirmation.
  • Make an export before bigger changes.

6. Generator handoff

The offline cleaning plan generator reads the planner JSON export. Do not edit the JSON manually before importing it into the generator.

  • Important fields such as cleaningSchedule, congregations, tasks and yearConfigs stay in the export.
  • Congregation IDs are included so names can be changed without breaking the link.
  • If the generator shows missing dates, export again from this planner after checking the selected year.
