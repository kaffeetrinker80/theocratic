🔒 Strict Privacy & Data Security 🔒 Strenge Privatsphäre & Datensicherheit Although these tools are hosted on GitHub, your data is never uploaded there. Whenever you open a file or import data, it is read strictly locally inside your own browser. Obwohl diese Tools auf GitHub gehostet werden, werden deine Daten niemals dorthin hochgeladen. Wann immer du eine Datei öffnest, wird diese streng lokal in deinem Browser gelesen.
📱 Tablet & Mobile Experience 📱 Tablet & Smartphone Nutzung You can get the best experience on a tablet by using your browser's zoom function (e.g., zoom out to 75% or 80%) to fit the interfaces perfectly. Auf dem Tablet erhältst du das beste Erlebnis, wenn du die Zoom-Funktion des Browsers nutzt (z.B. auf 75% oder 80% verkleinern), um die Oberflächen perfekt anzupassen.
