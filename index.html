Important note about data: This website itself does not include a public list of personal data and does not collect or store personal data on the server. These tools run directly in your browser. If someone uses a tool with names, assignments, notes, or other personal information, that data stays only on that device in the browser storage or inside a file the user explicitly imports or exports. Nothing is automatically uploaded to this website just by opening the page.
